> An official library for interacting with [CryptoBall Smart Contracts][smart-contracts-repo] via browsers over Web3 protocol. 

## Install

[npm][]:

```sh
npm install cryptoball-client
```

[yarn][]:

```sh
yarn add cryptoball-client
```
